Notion Guide

노션 포트폴리오에
GitGloss 위젯 넣기

4단계로 노션 페이지에 글래스모피즘 위젯을 삽입하세요.
포트폴리오, 학습 기록, 팀 협업 페이지 모두 가능합니다.

1
갤러리에서 템플릿 선택
GitGloss 갤러리에서 마음에 드는 위젯 템플릿을 클릭하세요. Stats 카드, 프로필 카드, 기술 배지 등 60개 이상의 템플릿이 있습니다.
💡 노션에는 프로필 카드 + 기술 배지 조합을 추천합니다
2
빌더에서 정보 입력
이름, 직함, 기술 스택을 입력하고 원하는 색상과 스타일을 선택하세요. 오른쪽 패널에서 실시간으로 미리보기가 업데이트됩니다.
💡 이모지 아바타를 선택하면 더 개성 있는 카드가 완성됩니다
3
HTML 탭의 코드 복사
코드 패널에서 HTML 탭을 선택하고 Copy 버튼을 클릭하세요. 노션은 Markdown이 아닌 HTML embed를 사용합니다.
HTML 
<img src="https://gitgloss.io/api/widget?style=glass&accent=4285F4
     &name=홍길동&role=Full-stack+Developer
     &badges=React,TypeScript,Node.js"
     alt="GitGloss Widget"
     style="max-width:100%;" />
4
노션 페이지에 임베드
노션 페이지에서 /embed 를 입력하고 복사한 이미지 URL을 붙여넣으세요. 또는 /image 블록에 URL을 직접 입력해도 됩니다.
💡 노션 갤러리 뷰에서 카드 커버 이미지로도 활용할 수 있어요
